알봄(Albom) 자주 묻는 질문

최종 수정일: 2026년 5월 3일

알봄을 이용해주셔서 감사합니다. 자주 묻는 질문을 모았습니다. 아래에서 답을 찾을 수 없다면 페이지 하단의 이메일로 문의해주세요.

로그인 및 계정

카카오 로그인이 안 돼요
다음 사항을 확인해주세요.
  1. 카카오톡 앱이 설치되어 있고 로그인되어 있는지 확인
  2. iOS 설정 → Safari → 쿠키 차단 해제 확인
  3. 인터넷 연결 상태 확인
  4. 알봄 앱을 완전히 종료한 후 다시 실행
계속 문제가 있다면 구글 로그인으로 시도해보시고, 두 방법 모두 안 된다면 아래 이메일로 기기 모델명과 함께 문의해주세요.
구글 로그인이 안 돼요
다음 사항을 확인해주세요.
  1. 구글 계정에 로그인되어 있는지 확인
  2. iOS 설정 → 알봄 → 권한 확인
  3. 네트워크 연결 상태 확인
  4. 앱 재시작
다른 기기로 데이터를 옮길 수 있나요?
네, 같은 소셜 계정(카카오 또는 구글)으로 로그인하시면 데이터가 자동으로 동기화됩니다. 별도의 백업 작업은 필요하지 않습니다.
회원 탈퇴는 어떻게 하나요?
앱 내 설정 메뉴에서 회원 탈퇴를 진행하실 수 있습니다.
  1. 앱 실행 후 우측 하단 설정 메뉴 진입
  2. 계정 관리 선택
  3. 회원 탈퇴 버튼 클릭
탈퇴 시 모든 개인 정보와 가족 정보, 영양제 기록이 즉시 삭제되며 복구되지 않습니다.

알림 관련

영양제 복용 알림이 오지 않아요
가장 자주 발생하는 문제입니다. 다음 순서대로 확인해주세요.
  1. iOS 설정 → 알림 → 알봄 → 알림 허용 확인
  2. iOS 설정 → 알봄 → 알림 → 사운드, 배지, 잠금 화면 모두 활성화
  3. 방해 금지 모드 또는 집중 모드가 켜져 있는지 확인
  4. 알봄 앱에서 영양제 알림 시간이 정확히 설정되어 있는지 확인
  5. iOS 백그라운드 앱 새로고침 활성화 (설정 → 일반 → 백그라운드 앱 새로고침)
모든 설정이 정상인데도 알림이 오지 않으면 앱을 완전히 삭제 후 재설치해주세요.
가족 알림(콕 찌르기)이 가족에게 전달되지 않아요
다음을 확인해주세요.
  1. 가족 구성원이 같은 가족 그룹에 속해 있는지 확인
  2. 가족 구성원의 기기에서 알봄 알림 권한이 허용되어 있는지 확인
  3. 가족 구성원이 알봄에 로그인되어 있는지 확인
  4. 네트워크 연결 상태 확인
알림 소리를 변경하고 싶어요
현재 버전에서는 iOS 기본 알림음을 사용합니다. 알림음 사용자 변경 기능은 추후 업데이트에서 검토 예정입니다.

가족 관리

가족 초대 코드는 어디서 찾나요?
앱 내 가족 메뉴에서 확인할 수 있습니다.
  1. 앱 실행 후 가족 메뉴 진입
  2. 가족 코드 영역에 6자리 코드 표시
  3. 해당 코드를 가족 구성원에게 공유
가족 초대 코드를 입력했는데 가족이 안 보여요
다음을 확인해주세요.
  1. 가족 코드를 정확히 입력했는지 확인 (대소문자 구분)
  2. 네트워크 연결 상태 확인
  3. 앱 재시작
  4. 가족 그룹 생성자가 본인을 가족에서 삭제하지 않았는지 확인
가족 구성원의 영양제 정보를 수정할 수 있나요?
가족 그룹 관리자는 모든 구성원의 영양제 정보를 추가, 수정, 삭제할 수 있습니다. 일반 구성원은 본인의 영양제 정보만 관리할 수 있습니다.
가족에서 나가고 싶어요
앱 내 가족 메뉴에서 가족 나가기를 선택하시면 됩니다. 가족에서 나가면 가족 구성원의 정보를 더 이상 볼 수 없으며, 본인이 입력한 영양제 기록은 유지됩니다.

영양제 관리

영양제를 어떻게 등록하나요?
  1. 홈 화면 우측 하단 추가 버튼 클릭
  2. 영양제 이름, 복용량, 복용 시간 입력
  3. 필요 시 사진 추가 (선택 사항)
  4. 저장 버튼 클릭
영양제 사진이 업로드되지 않아요
다음을 확인해주세요.
  1. iOS 설정 → 알봄 → 사진 접근 권한 허용
  2. iOS 설정 → 알봄 → 카메라 권한 허용
  3. 사진 파일 크기가 너무 크지 않은지 확인 (10MB 이하 권장)
  4. 네트워크 연결 상태 확인
복용 기록이 사라졌어요
다른 계정으로 로그인했을 가능성이 가장 높습니다. 영양제 등록 시 사용한 카카오 또는 구글 계정으로 로그인했는지 확인해주세요. 같은 계정으로 로그인하면 모든 데이터가 복구됩니다.
여러 영양제를 한 번에 등록할 수 있나요?
현재 버전에서는 영양제를 하나씩 등록해야 합니다. 일괄 등록 기능은 향후 업데이트에서 검토 중입니다.

AI 추천 기능

AI 추천은 어떻게 작동하나요?
사용자가 입력한 건강 관련 정보를 바탕으로 일반적인 영양제 정보를 제공합니다. AI 추천은 단순 참고용 정보이며 의료 진단을 대체하지 않습니다. 건강 관련 의사결정 시에는 반드시 의료 전문가와 상담하시기 바랍니다.
AI 추천이 작동하지 않아요
다음을 확인해주세요.
  1. 네트워크 연결 상태 확인 (AI 추천은 인터넷 연결 필요)
  2. 입력한 텍스트가 너무 짧거나 길지 않은지 확인
  3. 일시적인 서비스 점검 중일 수 있으니 잠시 후 재시도
AI가 추천한 영양제를 어디서 구매하나요?
앱 내 추천 결과의 링크를 클릭하면 외부 쇼핑몰로 이동합니다. 알봄은 단순히 정보를 제공하며, 실제 구매와 결제는 외부 쇼핑몰에서 진행됩니다.

개인정보 및 보안

제 정보는 안전하게 보관되나요?
모든 통신은 HTTPS로 암호화되며, 데이터는 Google Firebase의 보안 규칙으로 보호됩니다. 본인 또는 가족 그룹 구성원만 데이터에 접근할 수 있습니다. 자세한 내용은 개인정보처리방침을 확인해주세요.
제 데이터를 삭제하고 싶어요
회원 탈퇴 시 모든 데이터가 즉시 삭제됩니다. 회원 탈퇴 외에 특정 데이터만 삭제를 원하시면 아래 이메일로 문의해주세요.

기타

기능 추가 요청은 어디로 보내나요?
페이지 하단의 이메일로 보내주시면 검토 후 답변드립니다. 모든 의견을 소중하게 받아들이며, 다수의 사용자에게 도움이 되는 기능부터 우선적으로 개발할 예정입니다.
버그를 발견했어요
아래 정보와 함께 이메일로 보내주시면 빠르게 처리해드리겠습니다.
  • 기기 모델 (예: iPhone 15 Pro)
  • iOS 버전 (설정 → 일반 → 정보)
  • 알봄 앱 버전 (앱 내 설정에서 확인)
  • 버그 발생 상황 설명
  • 가능하면 스크린샷
유료 기능이 있나요?
현재 알봄의 모든 기능은 무료로 제공됩니다. 향후 프리미엄 기능이 추가될 수 있으나, 기본 영양제 관리 기능은 계속 무료로 유지될 예정입니다.
안드로이드 버전은 언제 출시되나요?
안드로이드 버전 출시 일정은 미정입니다. 출시 시 공지사항을 통해 안내드릴 예정입니다.
